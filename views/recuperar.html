Recuperar Clave Fiscal de AFIP

Blanqueo de clave fiscal por App Mi AFIP

  • Requisitos: DNI vigente, acceso a cámara frontal, acciones de prueba de vida (girar cabeza, pestañear, etc.).
  • Nivel recuperado: Se recuperará el mismo nivel de clave que se tenía previamente.
  1. Descargar la app Mi AFIP desde Play Store o App Store.
  2. Seleccionar la opción «Solicitar o recuperar clave fiscal» dentro de la app.
  3. Escanear el código de barras del DNI y completar captcha.
  4. Realizar prueba de vida enfocando el rostro según indicaciones.
  5. Crear y confirmar la nueva Clave Fiscal Nivel 3.

Recordatorio: Para Monotributo, se requiere Clave Fiscal Nivel 3.

Recuperar clave fiscal por cajero automático

  • Requisitos: Ser cliente de la red Banelco o Link y contar con clave de cajero.
  • Nivel recuperado: Se mantendrá el mismo nivel de clave que se tenía previamente.
  1. Acceder al cajero, ingresar tarjeta y PIN, y seleccionar «Gestión de claves».
  2. Seleccionar «Gestión Otras Claves».
  3. Seleccionar opción «02» para indicar recuperación de clave AFIP.
  4. Crear nueva clave de 8 dígitos evitando valores asociados, números iguales/consecutivos o fechas históricas. La clave tendrá validez de 48 horas.
  5. Antes de 48 horas, ingresar a la web de AFIP, colocar CUIT/CUIL y seguir las instrucciones para cambiar la clave.
  6. Crear nueva clave de 10 caracteres cumpliendo con requisitos de seguridad: mínimo 2 números, 4 letras, 1 minúscula y 1 mayúscula.

Recuperar clave fiscal en oficina de AFIP

  • Requisitos: Solicitar turno web, presentar formulario F.206, y documentación que acredite identidad según nacionalidad y residencia.
  • Nivel recuperado: Se mantendrá el mismo nivel de clave que se tenía previamente.

Presentarse el día del turno en la agencia seleccionada con la documentación y formulario para acreditar identidad y solicitar recuperación de clave.

Recuperar clave fiscal a través del Tótem

  • Requisitos: Tener datos biométricos registrados.
  • Nivel recuperado: Se mantendrá el mismo nivel de clave que se tenía previamente.

Dirigirse al puesto de autogestión disponible en algunas dependencias de AFIP, identificarse con huella digital y seguir las instrucciones del asistente virtual para recuperar la clave.

Recuperar clave fiscal por Homebanking

  • Requisitos: Ser cliente de un banco y contar con clave de Homebanking.
  • Nivel recuperado: Hasta Nivel 2.
  1. Ingresar al Homebanking del banco.
  2. Seguir indicaciones hasta encontrar «Servicios AFIP».
  3. Seleccionar «Blanqueo de Clave Fiscal».
  4. Ingresar nueva clave y repetirla para confirmación.
  5. Confirmación de transacción y obtención de Nivel 2.
